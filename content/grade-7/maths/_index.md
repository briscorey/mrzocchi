glossary:
  - topic: "Expressions"
    terms:
      - term: "Variable"
        emoji: "❓"
        definition: "A letter that represents an unknown number. Examples: x, n, a."
      - term: "Term"
        emoji: "🧱"
        definition: "A single part of an expression. In '3x + 5', the terms are 3x and 5."
      - term: "Coefficient"
        emoji: "🔢"
        definition: "The number in front of a variable. In '7x', the coefficient is 7."
      - term: "Constant"
        emoji: "🔒"
        definition: "A number with no variable attached. In '3x + 5', the constant is 5."
      - term: "Like Terms"
        emoji: "👯"
        definition: "Terms with the same variable and power. 3x and 7x are like terms. 3x and 5y are NOT."
      - term: "Simplify"
        emoji: "✂️"
        definition: "Make an expression shorter by collecting like terms. Example: 3x + 2x = 5x."
  - topic: "Equations"
    terms:
      - term: "Equation"
        emoji: "⚖️"
        definition: "A mathematical statement with an equals sign. Both sides must be equal. Example: 3x + 4 = 19."
      - term: "Solve"
        emoji: "🔓"
        definition: "Find the value of the unknown that makes the equation true."
      - term: "Inverse Operation"
        emoji: "↩️"
        definition: "The opposite operation. Addition ↔ subtraction. Multiplication ↔ division. Used to isolate the variable."
      - term: "Balance Method"
        emoji: "⚖️"
        definition: "Whatever you do to one side of the equation, you MUST do to the other side too."
  - topic: "Patterns & Functions"
    terms:
      - term: "Function Machine"
        emoji: "🏭"
        definition: "Takes an input, applies a rule, and produces an output. Input → Rule → Output."
      - term: "Common Difference"
        emoji: "📈"
        definition: "The amount the output changes when the input goes up by 1. This becomes the coefficient of n."
      - term: "Rule"
        emoji: "📝"
        definition: "The formula that connects input to output. Written as an + b. Example: 3n + 2."
  - topic: "Substitution"
    terms:
      - term: "Substitution"
        emoji: "🔄"
        definition: "Replacing a variable with a number, then calculating the result."
      - term: "BIDMAS / BODMAS"
        emoji: "📋"
        definition: "The order of operations: Brackets, Indices (powers), Division/Multiplication, Addition/Subtraction. Always follow this order!"

quiz:
  - q: "What is a variable?"
    a: "A letter representing an unknown number (e.g. x, n, a)."
  - q: "In the expression 5x + 3, what is the coefficient? The constant?"
    a: "Coefficient = 5. Constant = 3."
  - q: "Simplify: 4x + 7 + 3x - 2"
    a: "= 7x + 5 (collect like terms: 4x + 3x = 7x, 7 - 2 = 5)"
  - q: "Why can't you simplify 3x + 5 to 8x?"
    a: "3x and 5 are NOT like terms — one has a variable, the other doesn't."
  - q: "Solve: 4x + 3 = 23"
    a: "Step 1: 4x = 20. Step 2: x = 5. Check: 4(5) + 3 = 23 ✓"
  - q: "Solve: 2x - 7 = 11"
    a: "Step 1: 2x = 18. Step 2: x = 9. Check: 2(9) - 7 = 11 ✓"
  - q: "What is a function machine?"
    a: "Takes an input, applies a rule, produces an output. Input → Rule → Output."
  - q: "Outputs are 5, 9, 13, 17... What is the rule?"
    a: "Common difference = 4. When n=1: output=5. Rule: 4n + 1."
  - q: "If a = 5 and b = 3, find 2a + 3b."
    a: "2(5) + 3(3) = 10 + 9 = 19"
  - q: "If x = 4, find 3x² - 2."
    a: "3(4²) - 2 = 3(16) - 2 = 48 - 2 = 46"
  - q: "What does BIDMAS stand for?"
    a: "Brackets, Indices, Division/Multiplication, Addition/Subtraction — the order of operations."
  - q: "What is the balance method for solving equations?"
    a: "Whatever you do to one side, do to the other side. Keeps the equation equal."
